# Azure App Service用のスタートアップコマンド設定
# Azure App Serviceの設定で「スタートアップコマンド」に以下を設定してください：

# 方法1: 直接コマンドを指定
cd /home/site/wwwroot && npm start

# 方法2: カスタムスクリプトを使用
bash /home/site/wwwroot/.azure/startup.sh

# 方法3: 環境変数を使用
cd /home/site/wwwroot && PORT=$PORT npm start

# 注意: Azure App Serviceの設定で「スタートアップコマンド」フィールドに上記のいずれかを設定してください
