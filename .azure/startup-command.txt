# Azure App Service用のスタートアップコマンド設定
# Azure App Serviceの設定で「スタートアップコマンド」に以下を設定してください：

# 方法1: カスタムスクリプトを使用（推奨）
bash /home/site/wwwroot/.azure/startup.sh

# 方法2: 直接コマンドを指定
cd /home/site/wwwroot && bash .azure/startup.sh

# 方法3: 環境変数を使用
cd /home/site/wwwroot && PORT=$PORT bash .azure/startup.sh

# 注意: Azure App Serviceの設定で「スタートアップコマンド」フィールドに
# 上記のいずれかを設定してください
# 推奨は方法1です
